## 📌 Overview

A modern and responsive **Admin Dashboard** built using **HTML, CSS, and JavaScript**. The dashboard provides an intuitive interface for monitoring users, revenue, orders, and products while displaying analytics through interactive charts.

The project features a clean UI design with a sidebar navigation menu, animated statistic cards, real-time clock, dark/light theme toggle, notifications, and data visualization using Chart.js.

---

## ✨ Features

* 📊 Interactive Dashboard Analytics
* 👥 User Statistics Cards

* 🛒 Order Management Overview
